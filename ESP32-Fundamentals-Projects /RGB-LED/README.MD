# RGB LED Control using ESP32

## Overview

This project demonstrates how to control an RGB LED using an ESP32.

By controlling the Red, Green, and Blue channels individually, different colors can be produced using combinations of the three primary colors.

## Components Used

- ESP32 Development Board
- RGB LED (Common Cathode)
- Resistors
- Breadboard
- Jumper Wires

## Colors Displayed

The RGB LED is programmed to display:

- Red
- Green
- Blue
- Yellow
- Purple
- Cyan
- White

## Working

The Red, Green, and Blue pins of the RGB LED are connected to separate GPIO pins of the ESP32.

Since a Common Cathode RGB LED is used, the common pin is connected to GND.

Different colors are generated by turning different combinations of the Red, Green, and Blue channels ON or OFF.

For example:

- Red + Green = Yellow
- Red + Blue = Purple
- Green + Blue = Cyan
- Red + Green + Blue = White

## Concepts Practiced

- Controlling multiple GPIO outputs
- RGB color combinations
- `pinMode()`
- `digitalWrite()`
- Sequential LED control

## Learning Outcome

Through this project, I learned how an RGB LED works and how different colors can be generated by controlling its Red, Green, and Blue channels using the ESP32.
