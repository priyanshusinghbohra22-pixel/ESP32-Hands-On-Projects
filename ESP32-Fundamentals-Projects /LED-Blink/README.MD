# LED Blink using ESP32

## Overview

This is a basic ESP32 project that controls an LED using a GPIO pin.  
The LED is programmed to turn ON and OFF at fixed time intervals.

This project helped me understand the fundamentals of GPIO output control and basic ESP32 programming.

## Components Used

- ESP32 Development Board
- LED
- Resistor
- Breadboard
- Jumper Wires

## Concepts Practiced

- GPIO pin configuration
- `pinMode()`
- `digitalWrite()`
- `delay()`
- Basic breadboard connections

## Working

The LED is connected to an ESP32 GPIO pin configured as an output.

The program follows this sequence:

1. LED turns ON.
2. The program waits for a fixed delay.
3. LED turns OFF.
4. The program waits again.
5. The sequence repeats continuously.

## Learning Outcome

Through this project, I learned how to configure and control a GPIO pin on the ESP32. I also gained practical experience with breadboard connections and debugging basic wiring issues.
